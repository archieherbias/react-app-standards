# Creating your initial directories structure

### Initial Files and Directories for a React Web App

```bash
.eslintrc.json
.gitignore
jsconfig.json
package.json
yarn.lock
- public
- src
-- App.js
-- index.js
-- serviceWorker.js
-- setupTests.js
-- assets
---- images
---- fonts
-- components
---- index.js
-- constants
-- pages
---- index.js
-- redux
---- modules
------ index.js
---- store.js
-- routes
---- routes.js
-- services
---- api
---- data.services.js
---- storage.services.js
```
